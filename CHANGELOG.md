## [1.0.0] - 2026-03-01

### Added

- Initial release
- Adjust menu bar icon spacing via the `NSStatusItemSpacing` CFPreferences key
- Adjust menu bar selection padding via the `NSStatusItemSelectionPadding` CFPreferences key
- Live preview strip showing icon spacing in real-time
- Linked/unlinked padding mode (auto-follows spacing, or set independently)
- Apply changes with Control Center restart or full logout
- Restore system defaults with one click
- Haptic feedback on slider adjustments
- Automatic updates via Sparkle
